## Overview

Retrieve all trust centers or search by company name.

<ParamField query="search" type="string">
  Search query to filter trust centers by name (optional)
</ParamField>

## Interactive Testing

Use the playground below to test this endpoint directly in your browser.

<RequestExample>
```bash cURL
curl "https://trustlists.org/api/trust-centers?search=stripe"
```
</RequestExample>
