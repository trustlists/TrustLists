## GET /stats

Get aggregated statistics about the trust center directory.

### Request

```http
GET /api/stats
```

### Response

<CodeGroup>

```json Response
{
  "totalCompanies": 441,
  "generated": "2024-01-15T10:30:00Z"
}
```

```bash Example Request
curl "https://trustlists.org/api/stats"
```

</CodeGroup>

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `totalCompanies` | number | Total number of companies in directory |
| `generated` | string | ISO timestamp of when the response was generated |

### Example Usage

<CodeGroup>

```javascript JavaScript
// Get directory statistics
const response = await fetch('https://trustlists.org/api/stats');
const stats = await response.json();

console.log(`Total companies: ${stats.totalCompanies}`);
console.log(`Generated at: ${stats.generated}`);
```

```python Python
import requests

response = requests.get('https://trustlists.org/api/stats')
stats = response.json()

print(f"Total companies: {stats['totalCompanies']}")
print(f"Generated at: {stats['generated']}")
```

</CodeGroup>
