## GET /trust-centers

Retrieve trust center data with optional filtering.

### Request

```http
GET /api/trust-centers?search=stripe
```

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `search` | string | No | Search query to filter trust centers by name |

### Platform Values

Common platform values include:
- `SafeBase`
- `Conveyor`
- `Vanta`
- `TrustArc`
- `OneTrust`
- `Secureframe`
- `Whistic`
- `HyperComply`
- `Oneleet`
- `Self-hosted`
- `Other`

### Example Usage

<CodeGroup>

```javascript JavaScript
// Get all trust centers and filter by platform
const response = await fetch('https://trustlists.org/api/trust-centers.json');
const data = await response.json();

const vantaCompanies = data.data.filter(tc => tc.platform === 'Vanta');
console.log(`Found ${vantaCompanies.length} Vanta trust centers`);
vantaCompanies.forEach(tc => {
  console.log(`${tc.name}: ${tc.trustCenter}`);
});
```

```python Python
import requests

# Get all trust centers
response = requests.get('https://trustlists.org/api/trust-centers.json')
data = response.json()

# Filter by platform
vanta_companies = [tc for tc in data['data'] if tc['platform'] == 'Vanta']
print(f"Found {len(vanta_companies)} Vanta trust centers")
for tc in vanta_companies:
    print(f"{tc['name']}: {tc['trustCenter']}")
```
