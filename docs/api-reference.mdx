## Response Examples

### Trust Centers Response

<ResponseField name="data" type="array">
  Array of trust center objects
  
  <ResponseField name="name" type="string">
    Company name
  </ResponseField>
  
  <ResponseField name="website" type="string">
    Company website URL
  </ResponseField>
  
  <ResponseField name="trustCenter" type="string">
    Trust center URL
  </ResponseField>
  
  <ResponseField name="platform" type="string">
    Hosting platform (SafeBase, Conveyor, Vanta, etc.)
  </ResponseField>
  
  <ResponseField name="iconUrl" type="string">
    Company favicon URL
  </ResponseField>
</ResponseField>

<ResponseField name="meta" type="object">

  <ResponseField name="total" type="integer">
    Total number of trust centers
  </ResponseField>

  </ResponseField>
  
  <ResponseField name="version" type="string">

  </ResponseField>
</ResponseField>

## Usage Examples

<CodeGroup>
```javascript JavaScript
// Get all trust centers
const response = await fetch('https://trustlists.org/api/trust-centers');
const data = await response.json();

console.log(`Found ${data.meta.total} trust centers`);
data.data.forEach(tc => {
  console.log(`${tc.name}: ${tc.trustCenter}`);
});
```

```python Python
import requests

# Search for specific companies
response = requests.get('https://trustlists.org/api/trust-centers', 
                       params={'search': 'stripe'})
data = response.json()

print(f"Found {len(data['data'])} matching trust centers")
for tc in data['data']:
    print(f"{tc['name']}: {tc['trustCenter']}")
```

```bash cURL
# Get statistics
curl "https://trustlists.org/api/stats"

# Search for companies
curl "https://trustlists.org/api/trust-centers?search=vanta"
```
