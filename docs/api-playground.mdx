## Try the API

Use the interactive playground below to test TrustLists API endpoints directly in your browser.

<Note>
No authentication required - all endpoints are publicly accessible.
</Note>

## Available Endpoints

### GET /trust-centers

Retrieve all trust centers or search by company name.

**Query Parameters:**
- `search` (optional): Search query to filter trust centers by name

### GET /stats

Get statistics about the trust center directory.

<Tip>
The interactive playground will appear automatically once Mintlify processes the OpenAPI specification. You'll be able to test endpoints, see live responses, and copy working code examples.
</Tip>
