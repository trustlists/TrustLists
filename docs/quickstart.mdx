## API Access

TrustLists provides a simple REST API to access trust center data.

### Base URL

```
https://trustlists.org/api
```

### Authentication

No authentication required for public endpoints.

## Basic Usage

### Get All Trust Centers

<CodeGroup>

```bash cURL
curl "https://trustlists.org/api/trust-centers"
```

```javascript JavaScript
const response = await fetch('https://trustlists.org/api/trust-centers');
const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get('https://trustlists.org/api/trust-centers')
data = response.json()
print(data)
```

</CodeGroup>

### Get Trust Centers by Platform

<CodeGroup>

```bash cURL
curl "https://trustlists.org/api/trust-centers?platform=Vanta"
```
