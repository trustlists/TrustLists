## API Access

TrustLists provides a simple REST API to access trust center data programmatically.

<Warning>
**Important:** This is a static API served by GitHub Pages. Query parameters do not filter results server-side. All filtering must be implemented client-side.
</Warning>

### Base URL

```
https://trustlists.org/api
```

### Authentication

<Note>
No authentication required for public endpoints. This is a static API with no server-side rate limiting.
</Note>

## Basic Usage

### Get All Trust Centers

<RequestExample>
```bash cURL
curl "https://trustlists.org/api/trust-centers"
```
</RequestExample>

<ResponseExample>
```json Success
{
  "data": [
    {
      "name": "1Password",
      "website": "https://1password.com/",
      "trustCenter": "https://app.conveyor.com/profile/1password",
      "platform": "Conveyor",
      "iconUrl": "https://www.google.com/s2/favicons?domain=1password.com&sz=128"
    },
    {
      "name": "MongoDB",
      "website": "https://mongodb.com/",
      "trustCenter": "https://trust.mongodb.com/",
      "platform": "Self-hosted",
      "iconUrl": "https://www.google.com/s2/favicons?domain=mongodb.com&sz=128"
    }
  ],
  "meta": {
    "total": 441,
    "generated": "2024-01-15T10:30:00Z",
    "version": "1.0.0"
  }
}
```
</ResponseExample>

<CodeGroup>
```javascript JavaScript
const response = await fetch('https://trustlists.org/api/trust-centers');
const data = await response.json();

console.log(`Found ${data.meta.total} trust centers`);
data.data.forEach(tc => {
  console.log(`${tc.name}: ${tc.trustCenter}`);
});
```

```python Python
import requests

response = requests.get('https://trustlists.org/api/trust-centers')
data = response.json()

print(f"Found {data['meta']['total']} trust centers")
for tc in data['data']:
    print(f"{tc['name']}: {tc['trustCenter']}")
```
</CodeGroup>

### Filter Trust Centers by Search

<Tip>
TrustLists automatically detects hosting platforms using CNAME resolution and URL patterns. Common platforms include SafeBase, Conveyor, Vanta, and Self-hosted.
</Tip>

<RequestExample>
```bash cURL
curl "https://trustlists.org/api/trust-centers?search=vanta"
```
</RequestExample>

<ResponseExample>
```json Success
{
  "data": [
    {
      "name": "Linear",
      "website": "https://linear.app/",
      "trustCenter": "https://trustcenter.linear.app/",
      "platform": "Vanta",
      "iconUrl": "https://www.google.com/s2/favicons?domain=linear.app&sz=128"
    }
  ],
  "meta": {
    "total": 441,
    "generated": "2024-01-15T10:30:00Z",
    "version": "1.0.0"
  }
}
```
</ResponseExample>

<CodeGroup>
```javascript JavaScript
const response = await fetch('https://trustlists.org/api/trust-centers?search=vanta');
const data = await response.json();

const vantaCompanies = data.data.filter(tc => tc.platform === 'Vanta');
console.log(`Found ${vantaCompanies.length} Vanta-hosted trust centers`);
vantaCompanies.forEach(tc => {
  console.log(`${tc.name}: ${tc.trustCenter}`);
});
```

```python Python
import requests

response = requests.get('https://trustlists.org/api/trust-centers', 
                       params={'search': 'vanta'})
data = response.json()

# Filter by platform after getting results
vanta_companies = [tc for tc in data['data'] if tc['platform'] == 'Vanta']
print(f"Found {len(vanta_companies)} Vanta-hosted trust centers")
for tc in vanta_companies:
    print(f"{tc['name']}: {tc['trustCenter']}")
```
</CodeGroup>

### Client-Side Filtering

Since this is a static API, implement filtering in your application:

<CodeGroup>
```javascript JavaScript
// Fetch all trust centers
const response = await fetch('https://trustlists.org/api/trust-centers.json');
const data = await response.json();

// Filter by company name
const searchTerm = 'stripe';
const filtered = data.data.filter(company => 
  company.name.toLowerCase().includes(searchTerm.toLowerCase())
);

console.log(`Found ${filtered.length} companies matching "${searchTerm}"`);
```

```python Python
import requests

# Fetch all trust centers
response = requests.get('https://trustlists.org/api/trust-centers.json')
data = response.json()

# Filter by company name
search_term = 'stripe'
filtered = [
    company for company in data['data'] 
    if search_term.lower() in company['name'].lower()
]

print(f"Found {len(filtered)} companies matching '{search_term}'")
```
</CodeGroup>
