## GET /stats

Get aggregated statistics about the trust center directory.

### Request

```http
GET /api/stats
```

### Response

<CodeGroup>

```json Response
{
  "totalTrustCenters": 348,
  "platforms": {
    "Vanta": 45,
    "Drata": 32,
    "Secureframe": 28,
    "Oneleet": 15,
    "HyperComply": 12,
    "Self-hosted": 156,
    "Other": 60
  },
  "lastUpdated": "2024-01-15T10:30:00Z",
  "discoveryStats": {
    "lastDiscoveryRun": "2024-01-15T09:00:00Z",
    "newThisWeek": 8,
    "highConfidencePending": 3
  }
}
```

```bash Example Request
curl "https://trustlists.org/api/stats"
```

</CodeGroup>

### Response Fields

| Field | Type | Description |
|-------|------|-------------|
| `totalTrustCenters` | number | Total number of trust centers in directory |
| `platforms` | object | Count of trust centers by platform |
| `lastUpdated` | string | ISO timestamp of last data update |
| `discoveryStats` | object | Automated discovery statistics |

### Example Usage

<CodeGroup>

```javascript JavaScript
// Get directory statistics
const response = await fetch('https://trustlists.org/api/stats');
const stats = await response.json();

console.log(`Total trust centers: ${stats.totalTrustCenters}`);
console.log('Platform breakdown:');
Object.entries(stats.platforms).forEach(([platform, count]) => {
  console.log(`  ${platform}: ${count}`);
});
```

```python Python
import requests

response = requests.get('https://trustlists.org/api/stats')
stats = response.json()

print(f"Total trust centers: {stats['totalTrustCenters']}")
print("Platform breakdown:")
for platform, count in stats['platforms'].items():
    print(f"  {platform}: {count}")
```

</CodeGroup>
