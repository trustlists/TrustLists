## GET /trust-centers

Retrieve trust center data with optional filtering.

### Request

```http
GET /api/trust-centers?platform=Vanta&limit=10
```

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `platform` | string | No | Filter by hosting platform |
| `limit` | number | No | Number of results to return (default: 50) |
| `offset` | number | No | Number of results to skip (default: 0) |

### Platform Values

Common platform values include:
- `Vanta`
- `Drata` 
- `Secureframe`
- `Oneleet`
- `HyperComply`
- `Self-hosted`
- `Other`

### Example Usage

<CodeGroup>

```javascript JavaScript
// Get all Vanta-hosted trust centers
const response = await fetch('https://trustlists.org/api/trust-centers?platform=Vanta');
const data = await response.json();

console.log(`Found ${data.total} Vanta trust centers`);
data.trustCenters.forEach(tc => {
  console.log(`${tc.name}: ${tc.url}`);
});
```

```python Python
import requests

# Get first 20 trust centers
response = requests.get('https://trustlists.org/api/trust-centers', 
                       params={'limit': 20})
data = response.json()

for tc in data['trustCenters']:
    print(f"{tc['name']}: {tc['url']}")
```
